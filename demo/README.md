# Demo Folder

This folder contains demo materials for the Employee Task Tracker application.

## Contents

- **demo-video.mp4** - Main demo video (not tracked by git due to size)
- **demo-script.txt** - Script used for recording the demo

## Recording the Demo Video

See the detailed guide: [HOW_TO_RECORD_DEMO_VIDEO.md](../HOW_TO_RECORD_DEMO_VIDEO.md)

## Quick Steps:

1. **Record your screen** using:
   - Windows Game Bar (`Windows + G`)
   - OBS Studio (free)
   - Any screen recorder

2. **Save video here** as `demo-video.mp4`

3. **Upload to YouTube** or **GitHub** for sharing

4. **Update main README** with the video link

## Demo Script

Show these features in order:
1. Login page
2. Dashboard with statistics
3. Employee management (add, edit)
4. Task management (add, filter, update status)
5. Responsive design
6. Logout

Duration: 2-3 minutes
